Recover lost revenue with native WooCommerce emails and CRM syncing.

== Description ==

ReCart is an abandoned cart recovery plugin for WooCommerce. It automatically tracks abandoned checkouts, sends custom-branded recovery emails using WooCommerce's native engine, and syncs abandoned cart data directly to HubSpot.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/recart` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to WooCommerce > ReCart Settings to configure your recovery emails and CRM integration.
